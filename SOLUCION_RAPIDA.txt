╚════════════════════════════════════════════════════════════════════════════╝

🔧 PROBLEMA:
   El módulo de Mantenimientos no puede cargar porque los controladores 
   referencian tablas que no existen en las nuevas tablas con prefijos.

✅ SOLUCIÓN EN 2 PASOS:

┌─ PASO 1: Crear vistas en la base de datos ─────────────────────────────┐
│                                                                         │
│  Abre en tu navegador:                                                 │
│  http://localhost/digisports/public/setup_views.php                   │
│                                                                         │
│  Espera a ver: "✓ Todas las vistas se crearon correctamente."         │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─ PASO 2: Verificar que funciona ──────────────────────────────────────┐
│                                                                         │
│  Accede al módulo de Mantenimientos en la aplicación.                 │
│                                                                         │
│  Debería cargar sin el error 400.                                      │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
